### CUSTOMER CHURN PREDICTION & REVENUE FORECASTING


 Project Description

This project predicts customer churn and forecasts revenue using machine learning models. It includes EDA, preprocessing, classification, regression, and model interpretation using SHAP.

 Dataset

- Telecom customer dataset
- Includes customer demographics, services, and churn status

 Installation

git clone <your-repo-link>
cd teleconnect-ml-assignment
python -m venv venv
pip install -r requirements.txt

▶ How to Run

1. Run notebooks in order:
   - 01_EDA
   - 02_preprocessing
   - 03_classification
   - 04_regression
   - 05_interpretation

 Results

- Best Classification Model: Random Forest
- Accuracy: ~85% (example)
- Key Insight: Customers with high monthly charges and low tenure are more likely to churn

Tech Stack

- Python, Pandas, NumPy
- Scikit-learn
- SHAP
- Matplotlib, Seaborn
